crange

a range slider.


DESCRIPTION

When created, the range slider is stacked in the slider pane of the main Cecilia window in the order it is defined. The values of the range slider are passed to the orchestra with two variables. The range minimum is collected using gkvariable_namemin for a range slider whose -rate is k and givariable_namemin if the slider has a -rate value of i.  The range maximum is collected using gkvariable_namemax for a range slider whose -rate is k and givariable_namemax if the range slider has a -rate value of i. The -up flag passes the values of the range on mouse-up if set to 1 or passes on mouse-down if set to 0. The -gliss flag determines the duration of the portamento (in sec).  The resolution of the range slider can be set to int or float using the -res flag. Slider colour can be set using the -col flag and a colour argument. 

Every time a range slider is defined, two graph line are automatically defined for the grapher in the Cecilia grapher pane. One is linked to the minimum value of the range, the other one to the maximum value of the range. The recording and playback of an automated slider is linked to its graph line. For further informations, refer to the Slider section.

  NOTE: in order to quickly select the minimum value (and graph line), you can click on the left side of the crange label, to select the maximum value (and graph line), click on the right side of the label.


INTERFACE SYNTAX

crange

(compulsory)

variable_name (mandatory, no spaces allowed)

-label Text showing in Cecilia slider pane (spaces allowed). Default value = ''

-min Minimum value (numerical). Default value = 0

-max Maximum value (numerical). Default value = 1

(optional)

-init Pair of initial range values (numerical) separated by a comma. Default value = min value, max value

-unit Unit symbol showing in Cecilia slider pane (text). Default value = ''

-res Slider resolution {float, int}. Default value = float

-rate Update rates of the csound value {k, i}. Default value = k

-gliss Portamento between values in seconds. Default value = 0.025

-rel Slider scaling {lin, log}. Default value = lin

-up Value passed on mouse up (0 = on mouse down, 1 = on mouse up). Default = 0

-midictl MIDI controller number assigned to the slider (numerical). Default value = -1 (None)

-col Colour of the slider (see Colour arguments). A default colour is applied by Cecilia

-help Text for tooltips (space allowed) Default value = ''


ORCHESTRA SYNTAX

krandomFreq  random  gkfreqmin, gkfreqmax

 

gkfreqmin and gkfreqmax would be the minimum and maximum values directly obtained from a range slider whose variable name is respectively freq with a -rate of k.


EXAMPLES

crange freq -label Pitch range -unit pch -min 1 -max 90 -init 40,80  -col red

crange speed -label Speed range -unit Hz -rel log -min 0.01 -max 100 -init 1, 10 -col green

crange dur -label Duration range -min .005 -max 5

crange intensity -label Intensity range -unit db -min 30 -max 90 -init 50.0, 80.0 -col orange \\

-help Range of intensity of the grains
