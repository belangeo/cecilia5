Graph


The graph area is in the center of Cecilia's interface. It presents a serie of lines that can be modified with the graph tools. Those lines are linked to parameters of the Cecilia modules.




The graph's horizontal axis is always the performance time. Vertical axis automatically reajusts to the selected line.


Note that the Undo action is always available for graph manipulations using Command+Z or the through the Edition Menu



Line selector

To work on a line, you must first select it. To do so, use the line selector in the upper left corner of the graph:



By selecting a line in this popup menu, you can subsequently work with it in the graph. A graph line can also be selected by clicking on its associated slider label, if it exists.



Line Save/Load

A selected line can be imported or exported using the following tools:



- 1st icon (floppy disk) allows to save the line currently being worked on.

- By clicking the 2nd icon (a folder), a previously saved* graph line can be loaded.

- The 3rd icon reinitializes the line to it's default value.

- The 4th icon (an eye) hides/shows the line in the graph region. Please note, hiding a line does not disable its functionality.


* A line is saved with absolute values. This allows you to replace a line that has a different range of values.


Line

A graph line is a two dimensional representation of a time variant value. It shows the value of the widget (Y-axis) in function of the performance time (X-axis). In other words, the line shows the evolution of a parameter through performance duration.



A line is scaled to the total duration of a performance. If you modify the performance duration, the lines will be  compressed or expanded to match it. 


A graph line is made out of two elements: points and line segments. Points are the element that the user can control. Line segments are automatically reajusted to connect points. Line segments can be straight of curved using the arrow tool.



Graph Tools

A selected line can be modified with the mouse using one of the four graph tools:



The arrow tool allows the manipulation of line points:

1) A point can be moved by clicking and dragging it.

2) A new point can be created by double-clicking anywhere in the graph region

3) To delete a point, CTRL + click OR right click on it. One or many points can be selected and erased with the DELETE key.

4) To select a single point (for further manipulation), click on it once

5) To select multiple points, SHIFT + click on any points. You can also click in an empty region and drag to create a selection region (marquis). All points in this region will be selected.

6) To move a selection, click and drag any of the points selected


When the arrow tool is over the selected line, this line turns black. Double-clicking on it will alternate between two types of point connections. First type is line segments, second type is smooth curves. Click and dragging a blackened line allows you to vertically move the whole line


The pen tool allows you to draw line points:

1) To draw a single point, click in the graph area where a point is desired.

2) To draw multiple points, click and drag the pen tool. Please note, the pen tool overwrites points. Also, the number of points drawn depends on the speed of the movement.


The magnifier tool allows to zoom on line points (and thus have more precision):

1) To zoom in with the magnifier tool, click and drag over the desired region. The region created by the click-drag action defines the new view of that that area.

2) To reset the zoom level, hit the ESCAPE key.


The hand tool allows moving around in a zoomed-in graph:

In a zoomed-in graph, 

1) click and drag to move around.

2) click on a point to set it at the origin of the graph view.


Line generators

Cecilia's graph also include a series of generators to create, modify and randomize a selected line. When choosing one of the generators, a frame opens to set the generator values. If Apply is pushed, the selected line will be modified. When hitting the Close button, the frame closes without modifying the selected line.



The stochastic generator creates lines using various random distributions:

1) Select the random distribution (browse the web to see the characteristics for each distribution)

2) Select the interpolation. Linear will connect random points with line segments. Sample hold will create instant transitions between random values.

3) Determine how many points are desired using the Points slider

4) Determine the minimum and maximum values of the random distribution using the Min and Max sliders. Note that min and max values will be rescaled to match the Y-axis scale of the graph.

5) Some random distributions need additions variables. When available, specify them with the x1 and x2 sliders.



The waveform generator creates lines with predefined shapes:

1) Select the shape between Sine, Square and Sawtooth

2) The Points slider determines the number of points that will shape the line

3) The Amp slider determines the amplitude of the shape, 1 being the full range of the Y-axis.

4) The Freq slider sets the number of time a shape is repeated over the X-axis of the graph.

5) The Phase slider sets the the starting point in the shape.

6) When the shape is set to Square, the Width slider sets the time ratio of each alternating values. For example, when the Width slider is set to 0.25, the first state occupies 25% of the whole wave form.



The function processor modifies the selected line in various ways:

1) Select the processor to apply to the selected line.

2) Scatter moves the points around randomly within an independant X (Scatt X slider) and Y (Scatt Y slider) range.

3) Jitter creates a new line approximated from your original line. Number of points in that line is set with the Points slider. Jitter in X is set using the Jitt X slider and in Y using the Jitt Y slider.

4) Comp/Expand function is used to compress or expand the range of an existing line. Comp X slider changes the ratio of time between each points. The range of values can the modified using the ratio of the Comp Y slider. A line can also be moved horizontally with the Offset X slider and vertically with the Offset Y slider.
