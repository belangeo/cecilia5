Main Control Panel


Cecilia's performance can be lauched using the Main Control panel.



Cecilia allows two types of performances: Realtime and Rendering. Click on the play or the record button will generate either a realtime or a rendering performance. 



An indication of the Performance Time can always be found in the Main Control Panel.



Realtime performance


By clicking on the play button, the user lauches a realtime performance. The Play buttons immediately turns into a Stop button:



During a realtime performance, the user can hear the sound produced by the Cecilia module. The user can then tweek this sound by playing around with the various widgets of the module. He can record automations, listen to the modifications he had previously made to a graph line. At any time, a realtime performance can be stopped by clicking on the stop button. 


NOTE: Shortcut for the Play/Stop action is Cmd + . 

(yes, you read correctly, cmd + [dot])


In a realtime performance, the sound produced by the Cecilia module is outputed to the Digital-to-Analog Converter (DAC). The DAC is defined in the Audio tab of the Preference Panel.



Rendering performance


Once the user is satisfied with the sound, the result can be dumped in a soundfile. To do so, the user starts a rendering performance by clicking on the Rec button in the Main Control Panel.

The Rec button turns red and the Play button turns into a Stop button:



At this point, Cecilia will render the performance a quickly as possible. The output sound will be dumped in a sounfile defined in the Output Panel. The caracteristics of this soundfile can be defined in the Audio tab of the Preference Panel.



Performance Time


A Performance Time indicator has been added to the Main Control Panel so that the user can easily know where in the performance he is located. 



The timer is divided in minutes, seconds and centi-seconds.
