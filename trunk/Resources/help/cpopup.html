cpopup

a popup (contextual) menu.


DESCRIPTION


The popup menu appears in the lower left corner of the interface in the order listed. The selected value of the popup menu is passed to the orchestra with a variable named gkvariable_name or givariable_name. The value is a number representing the order of the choice in the menu. Values are between 0 and (number of choices) - 1. The menu can be updated at k or i rate using -rate flag. To define the choices in the menu, list the various options after the -value flag. The -init flag sets the initial numerical value of the popup menu. Popup menu colour can be set using -col flag and a colour argument. 


NOTE Changing the value of a k-rate popup menu in a performance can cause pops and clicks.


INTERFACE SYNTAX


cpopup

(compulsory)

variable_name (mandatory, no spaces allowed)

-label Text showing in Cecilia slider pane (spaces allowed). Default value = ''

-value List of the choices in the popup menu

(optional)

-rate Update rates of the csound value {k, i}. Default value = k

-init Initial value (numerical). Default value = 0

-col Colour of the slider (see Colour arguments). A default colour is applied by Cecilia

-help Text for tooltips (space allowed) Default value = ''


ORCHESTRA SYNTAX


kcompratio table gkcompratio, 1

 

where gkcompratio would be a table index associated to a value usable in the orchestra


EXAMPLES


cpopup compratio -label Compress Ratio  -value 1:1 2:1 3:1 5:1 8:1 13:1 21:1 34:1 55:1 inf -init 3 -col olivegreen

cpopup waveform -label Waveform -value Sin Triangle Square UpRamp -rate i -init 2 -help Select a waveform for the LFO
