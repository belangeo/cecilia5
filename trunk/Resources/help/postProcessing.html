Post-Processing Panel


New to version 4.03 of Cecilia, the Post-Processing panel is a good way of adding processing to the signal coming out of a Cecilia module. This way, the user can easily insert up to three processing modules at the end of the signal chain. The Post-Processing panel has a series of pre-programmed effects that can be controlled with three simple dial buttons.Â 



The signal coming out of the Cecilia module is passed through the Post-Processing unit before being outputed. The three inserts are serialized from the top to the bottom. An insert consists of three dials, a context menu to choose the effect and a second context menu to choose the type of effect:




Insert


For each insert, the user can select the effect he wants to apply. To do so, he has to browse the Effect context menu:



Clicking on an effect will load it. Each dial will be named after the parameter it controls. The Type context menu will feature a group of options to configure the selected effect.
