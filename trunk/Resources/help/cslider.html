cslider

an horizontal slider.


DESCRIPTION

When created, the slider is stacked in the slider pane of the main Cecilia window in the order it is defined. The value of the slider is passed to the orchestra with a variable named gkvariable_name for sliders whose -rate is k and with a variable named givariable_name if the slider has a -rate value of i.  The -up flag passes the value of the slider on mouse-up if set to 1 or passes on mouse-down if set to 0. The -gliss flag determines the duration of the portamento (in sec).  The resolution of the slider can be set to int or float using the -res flag. Slider colour can be set using the -col flag and a colour argument. However, sliders in i-rate are greyed and the -col flag is ignored.

Every time a slider is defined, a corresponding graph line is automatically defined for the grapher in the Cecilia grapher pane. The recording and playback of an automated slider is linked to its graph line. For further informations, refer to the Slider section.


INTERFACE SYNTAX

cslider

(compulsory)

variable_name (mandatory, no spaces allowed)

-label Text showing in Cecilia slider pane (spaces allowed). Default value = ''

-min Minimum value (numerical). Default value = 0

-max Maximum value (numerical). Default value = 1

(optional)

-init Initial value (numerical). Default value = min value

-unit Unit symbol showing in Cecilia slider pane (text). Default value = ''

-res Slider resolution {float, int}. Default value = float

-rate Update rates of the csound value {k, i}. Default value = k

-gliss Portamento between values in seconds. Default value = 0.025

-rel Slider scale {lin, log}. Default value = lin

-up Value passed on mouse up only (0 = false, 1 = true). Default = 0

-midictl MIDI controller number assigned to the slider (numerical). Default value = -1 (None)

-col Colour of the slider (see Colour arguments). A default colour is applied by Cecilia

-help Text for tooltips (space allowed) Default value = ''


ORCHESTRA SYNTAX

asig oscil gkamp, gifreq,1

 

gkamp and gifreq would be values directly obtained from two sliders whose variable names are respectively amp with a -rate of i and freq with a -rate of k.


EXAMPLES

cslider gain -label Makeup Gain -min 0.5 -max 100 -init 1 -rel log -col olivegreen -help Be careful when using with "Restore Amp"

cslider num -label Number of Layers -rate k -res int -min 1 -max 20 -init 5 -gliss 0 -up 1 -help No. of cascaded filters

cslider freq  -label Center Q  -min 0.05 -max 3.0

cslider pitch -label Base pitch -min 10 -max 10000 -init 1 -rel log -init 500 -gliss .01 -up 1 -col marineblue //

                                                        -help Move this slider to make pitch rise or fall
