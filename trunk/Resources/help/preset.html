Presets Panel


The parameters of a module can be saved using the Preset function:




The user can have many presets within a module (the number is virtually illimited):



Each preset holds the current values of all the widgetsimn a module (including Post-Processing Units). This way, the user can easily and quickly go from one state to the other.Â 


The Preset panel has two tools to control the various presets:



To save a preset, hit the floppy disk icon in the preset tools. The user has to give the preset a name. Values of all the widgets will be saved inside this named preset. Hitting the X icon will delete the currently loaded preset, after the user confirms his action.



The preset menu always contains an Init preset. When selected, this preset resets the interface to its original values. Doing so does not delete the saved presets.The init preset cannot be deleted
