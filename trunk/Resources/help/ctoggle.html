ctoggle

a checkbox.


DESCRIPTION

The toggle appears in the lower left corner of the interface in the order defined. The value of the toggle is passed to the orchestra with a variable named gkvariable_name or givariable_name. The value of the variable is 0 if unchecked and 1 if checked.The toggle value can be updated at k or i rate using -rate flag. The -init flag sets the initial value of the toggle (0 or 1). Toggle colour can be set using -col flag and a colour argument. 


NOTE Changing the value of a k-rate toggle in a performance may cause pops and clicks.


INTERFACE SYNTAX

ctoggle

(compulsory)

variable_name (mandatory, no spaces allowed)

-label Text showing in Cecilia slider pane (spaces allowed). Default value = ''

(optional)

-rate Update rates of the csound value {k, i}. Default value = k

-init Initial value (0 or 1). Default value = 0

-col Colour of the slider (see Colour arguments). A default colour is applied by Cecilia

-help Text for tooltips (space allowed) Default value = ''


ORCHESTRA SYNTAX

if gkonOff==1 then

...

endif

 

where gkonOff is a switch allowing a process or not


EXAMPLES

ctoggle 01resonOn -label Resonator 01  -init 1 -rate k -col olivegreen -help Turn on/off Resonator 1
