If there is an older installation of Cecilia 4 on the system, 
it is recommended to uninstall it first.

To use the binary distribution of Cecilia 4, you need to install 
Csound 5.12.1 (double version) on your system.
 
Csound (Csound5.12.1-gnu-win32-d.exe) can be downloaded on sourceforge:

http://sourceforge.net/projects/csound/files/csound5/csound5.12/
